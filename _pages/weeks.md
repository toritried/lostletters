<h1>Welcome to My Digital Hobonichi Weeks</h1>
This page is a <em>digitally-enhanced</em> version of my memory keeping spreads from my Hobonichi Weeks. The entries are pretty similar in contents to what I write down with fountain pens, but they contain hyperlinks, pictures, and additional space that I'm not afforded in the analog version. I always like to include a daily memory, which I've highlighted, to help me recall the things I've experienced in this fleetingly brief life. These are all the mundane moments that don't make it into things like mastodon posts, yet feel much more interesting and intimate in some way.

<h2>Credits & Final Thoughts</h2>
I wanted try creating a responsive, column-height-balancing flexbox layout inspired by <a target="_blank" href="https://repth.neocities.org/openbook">Repth's open book tutorial</a>, <a target="_blank" href="https://itinerae.neocities.org/">itinerae's journal</a>, and my Hobonichi weeks. Unfortunately, this layout only looks okay in the sakura theme for my website because the variable colors are kind of whack in dark layouts. Also, I don't think it's possible to style the flexbox gap enough to make the illusion of a book fold in the middle of the desktop layout without breaking the layout I already have. Perhaps there's something I could do with a styled border that is offset from a side and a media query to make it go away when in a single-column layout, but I can't be bothered. Tinkering with the headings to get something I liked took the longest by far, and the CSS is an absolute mess. Updating a whole week in here takes forever, so I'm not sure I'll keep adding more entries, but this was a fun coding skill challenge.
